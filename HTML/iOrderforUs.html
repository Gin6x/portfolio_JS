i_OrderforUs

Easy orders recording and notify your work companions

This app is design for people like me, who always draw the short straw and do the ordering for everyone. This app allow me to record every order, especially for takeaway ones and get my colleagues notify by sending them a email through the app, onto their company's email. They know exactly what they ordered even if they are busy and who to pay back.

    Technologies

  • UIKit
  • User Defaults
  • MessageUI
  • Swift

Create

Upload image of the menu or receipt using the camera button.
Input all order for each individuals using the add item button.

Edit

Tap on the order box to edit each item.
User could also swipe left to directly delete the item.

History

User could check the previous order in the history page.
While long-pressing the order box, a pop-up menu from the bottom to let user to reorder the same order.
Greatly lesson the hassle to re-type everything from stretch.

Mail

After pressing save on the order page, a mail composer would pop-up and the template with order detail written.
